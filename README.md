# Customer Relationship Management (CRM) System

A web-based Customer Relationship Management (CRM) application developed using **PHP**, **MySQL**, **HTML**, **CSS**, and **JavaScript**. The system helps organizations manage customers, employees, products, and orders through a simple and user-friendly interface.

## Features

- Secure Admin Login
- Dashboard with Business Statistics
- Customer Management
- Employee Management
- Product Management
- Order Management
- Reports and Analytics
- Global Search
- User Profile Management
- Activity Log

## Technology Stack

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- XAMPP

## Installation

1. Clone the repository.
2. Import `database.sql` into MySQL.
3. Update database credentials in `config.php`.
4. Start Apache and MySQL using XAMPP.
5. Open:

```
http://localhost/crm/login.php
```
